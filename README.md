# Sabbir Uddin Ahmed Portfolio

Premium modern personal portfolio website for Sabbir Uddin Ahmed, a Strategic Planner, Event & Activation Specialist, and Brand Marketing Professional from Bangladesh.

## Files

- `index.html` - Main website markup
- `style.css` - Responsive visual design, glassmorphism, layout, and animations
- `script.js` - Typing effect, counters, slider, lightbox, scroll progress, and UI interactions
- `assets/` - Local placeholder image and document folder

## How To Run

Open `index.html` directly in a browser.

For a local server, run one of these from the project folder:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
